Site Overview

This site was built using the Angular.js single page application (SPA) framework. Upon initial display, index.html is loaded along with all its Javascript dependencies. As you navigate from page to page, index.html remains as the underlying framework for the page, but the content (html templates, images, etc...) is loaded dynamically via Ajax calls, all of which is handled by Angular and its routing methods.

GitHub

All of the code for this site can be viewed on GitHub. I develop locally on my home Windows machine and deploy using Git to my Linux web host. So what you see here on techmeholmes.com as well as anything in the Samples links should match exactly to what's in my GitHub repos.

Nerd-oriented content ahead!


This entire site is built around the Angular.js SPA (Single Page Application) framework. The page header and footer, the images blocks (with pop-up captions) on the "About Me" page, even this list of site features, are all custom directives. Arguably the most powerful tool within the Angular.js framework, they make the page markup immensely easier to read as well as promoting code reuse.
Bootstrap
The Bootstrap grid system is used to allow this site to expand and contract to nearly any viewport size while maintaining readability.
Though the "Angular way" professes little to no need for direct DOM manipulation or jQuery effects, I tend to disagree on occasion. I still find animation effects (such as the caption slide up and down on an image) to be easily accomplished with jQuery, while the same effect using Angular animation tactics is tedious and overly laborious.
